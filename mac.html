kill a user session

Even if you don't know another user's password you can close a session they left open on your machine:

  1. Open Activity Monitor
  2. In the CPU tab, Process Name column select a loginwindow process with the user you want to kick.
  3. alt+command+q

local web server

  1. Start the local Apache web server: sudo apachectl start
  2. In a browser's address bar, enter: localhost. If it's working the message it works is displayed.
  3. Create a website at /Library/WebServer/Documents.
