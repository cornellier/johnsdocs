Run Sublime Text 2 from the command line on macOS

Sublime Text 2 includes a CLI (command line interface) which can open Sublime, for example: /Applications/Sublime\ Text.app/Contents/SharedSupport/bin/subl &

Since that's not easy to type, make a symlink in a directory in $PATH to subl: sudo ln -s /Applications/Sublime\ Text.app/Contents/SharedSupport/bin/subl /usr/local/bin/subl

and test: subl foo.txt

The file argument can take multipe files.
