git

Some commands I use in git

vocabulary

connect to an existing repo

git clone https://github.com/cornellier/johnsdocs.git

create and add a file

echo .DS_Store >> .gitignore
git add .gitignore

commit (aka stage)

git commit .gitignore -m "add .DS_Store"

diff with origin

git fetch origin
git diff origin -- [local-path]

push to the remote repo

git push
init
touch README
add README
commit -a -v -m 'message'
push
pull
fetch (what is the difference between pull and fetch?)

To undo git add before commit git reset [file]. This removes it from without changing anything else.

To ammend the most recent commit message do git commit --amend

to undo a commit and redo:

git commit -m "oops"
git reset HEAD~
[ edit files ]
git add ...
git commit -c ORIG_HEAD
